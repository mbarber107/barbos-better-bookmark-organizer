HOW TO USE AUTO-TAG
================================================================================

1. Open the extension manager (click extension icon â†’ "Open Manager")
2. Click "Auto-Tag" in the sidebar (ðŸ¤– icon)
3. View your statistics:
   - Total bookmarks
   - Already tagged
   - Untagged  
   - Can be auto-tagged

4. Choose scan type:
   - "Scan Untagged Bookmarks" - Only bookmarks without tags
   - "Scan All Bookmarks" - Get suggestions for everything

5. Review suggestions:
   - Each card shows bookmark info, current tags, and suggestions
   - Check/uncheck tags you want to apply
   - See folder suggestions

6. Apply tags:
   - "Apply Selected Tags" - Apply to one bookmark
   - "Apply All Suggestions" - Tag everything at once

7. Done! Your bookmarks are now organized with meaningful tags
